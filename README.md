# Python Data Visualization – Taxis Dataset

## Project Description
The objective of this project is to explore taxi trip data and visualize patterns such as fare trends, trip distribution, and relationships between variables. he project includes data loading, data cleaning (handling missing values), and multiple visualizations created using Matplotlib and Seaborn.

---

## Project Objectives
- Load and explore the taxis dataset
- Handle missing values using appropriate techniques
- Perform data preprocessing
- Create multiple visualizations to analyze the data
- Understand relationships between variables such as fare, distance, tips, and payment method

---

## Dataset
The dataset is loaded directly using Seaborn

---
## Visualizations Created

### Matplotlib Visualizations
- **Line Chart** – Fare over time  
- **Bar Chart** – Total fare by pickup borough  
- **Pie Chart** – Payment method distribution  
- **Histogram** – Distance distribution  
- **Box Plot** – Tip distribution by borough  

### Seaborn Visualizations
- **Count Plot** – Trips per pickup borough  
- **Scatter Plot** – Distance vs Fare  
- **Heatmap** – Correlation between numerical variables  
- **Pair Plot** – Relationship between distance, fare, tip, and total  
- **Violin Plot** – Fare distribution by payment method  

---

## Key Insights
- Fare generally increases with travel distance.  
- Certain boroughs have more trips than others.  
- Payment methods vary across trips.  
- Tips vary depending on borough and trip distance.

This project demonstrates how Python can be used for effective data analysis and visualization.
